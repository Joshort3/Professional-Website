Arduino Robotic Arm

I built a Arduino Robotic Arm and I am now in the process of programming the arm to write in English. The inputs for the arm will be keyboard inputs, which will decide what letter to write in what order. For example, typing the word ‘hello’ will output the Arm to write the word ‘hello’ on paper. Traffic Light System

Using my knowledge of circuits and programming, I created a traffic light system on a breadboard. The code behind the system consist of a single while loop and an interrupt handler for a button press. The light system is powered by a Raspberry Pi, so the main library used is RPi.GPIO.

Turn-based Role-Playing Game

I am currently developing a two-dimensional game similar to the games in the Fire Emblem Franchise. Like the Fire Emblem games, there are going to be units/characters with classes and certain stats such as health points, strength, defense, etc. Players will get to control these characters like an army and will engage will enemy CPU armies. Defeating enemy units will level-up characters to prepare for future levels of the game. I going to be designing my own characters, classes, and stages.
